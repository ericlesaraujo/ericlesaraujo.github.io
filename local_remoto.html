Como enviar o projeto local para um repositório remoto

Depois de criar o repositório no github, eu realizei os comando acima através do gitbash no meu diretório local. Depois de dar o commit, eu executei os comandos abaixo:

a) git remote add origin https://github.com/nomegit/repo
b) git push -u origin master
e) git clone https://github.com/nomegit/repo: após transformar o diretório local com o "git init", você pode utilizar esse comando para baixar os arquivos de um repositório remoto.
f) git push https://github.com/nomegit/repo: depois de concluir o desenvolvimento, é necessário usar os comandos "git add" e "git commit", mas as alterações ainda não serão enviadas para o github. Isso só será possível depois de fazer o push (empurrar).
