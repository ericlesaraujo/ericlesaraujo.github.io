O Github

Crie uma conta no GITHUB. Em termos simplificados, ele permite guardar todos os seus projetos na nuvem. Outros desenvolvedores também poderão vê-los e contribuir para melhoria do seu código.

Muitas empresas de tecnologia pedem o GITHUB do candidato como se fosse um portfólio. Acesse e crie a conta clicando em Sign up.
