O Github

Crie uma conta no GITHUB. Em termos simplificados, o github é uma rede social que possibilita guardar projetos em nuvem. Outros desenvolvedores podem vê-los e com sua permissão, eles podem até contribuir para melhorar seu código.

Muitas empresas de tecnologia pedem o GITHUB do candidato como se fosse um portfólio. Acesse [https://github.com/] e crie a conta clicando em Sign up.
