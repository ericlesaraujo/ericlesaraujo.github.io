Comandos básicos

a) git init: transforma a pasta atual em um repositório local.
b) git status: mostra informações do repositório local (commits pendentes, arquivos não monitorados pelo git, etc...).
c) git add nomearq.txt || git add . || git add pasta: permite que o git coloque os arquivos em um modo de espera para commit. Esse comando é necessário antes de efetuar um commit.
d) git commit -m "MENSAGEM": efetua um commit.

Criação do arquivo .gitignore:

Você também pode criar um arquivo chamado ".gitignore" no seu repositório local. Coloque lá dentro os nomes de todos os outros arquivos que o git precisa ignorar na hora de adicionar e comitar.
