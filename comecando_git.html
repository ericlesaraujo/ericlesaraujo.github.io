Começando a usar GIT

Instale um programa chamado GITBASH [https://git-scm.com]. Clique em Downloads. Na próxima tela clique no link que corresponde ao seu sistema operacional.

O download será feito automaticamente quando você escolher o S.O e for redirecionado de página.

Caso não aconteça o download automático, você mesmo precisará selecionar a versão do GITBASH de acordo com o seu sistema operacional.

É muito fácil, basta ler atentamente as instrunções. O meu S.O, por exemplo, é Windows 64 bits.
